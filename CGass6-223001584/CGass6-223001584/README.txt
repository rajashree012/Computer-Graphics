1. '+' - it does both linear subdivision and averaging
2. 'L' - it does only linear subdivision
3. 'A' - it does only averaging
4. 'Left mouse button' - If it is dragged left then it rotates about y-axis in clockwise direction 
						 If it is dragged right then it rotates about y-axis in anticlockwise direction
						 If it is dragged up then it rotates about x-axis in clockwise direction
						 If it is dragged down then it rotates about x-axis in clockwise direction
5. 'Right mouse button' - If dragged up the image gets enlarged
						  If dragged down the image diminishes
